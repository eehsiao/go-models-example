# go-models-example
example that use [go-models](https://github.com/eehsiao/go-models)

# Docker
Easy to start the test evn. That you can run the example code.
```bash
$ docker-compose up -d
```

# Build
in your `go-models-example` folder, and build and install into ~/go/bin
```
go install ./example1
```

# run
run in ~/go/bin
```
./example1
```
